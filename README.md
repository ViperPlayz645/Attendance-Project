The attendace project is designed to allow board members to gather useful data in term of who shows up, how many show up and what they do during club.

The positon info is gathered by the back end google script which connects to a google sheets which is populated based on the sheet provided by TooCool for those who pay their club dues.
In this repository you will find the HTML font end as well as the link to the google script + sheet. In the HTML front end you will find the options that are offered in the drop down along side any other responses / feedback comments made to the user.
